- Người mới học lập trình muốn tìm hiểu Rust
- Lập trình viên đã biết ngôn ngữ khác (C/C++, Python, Java...) muốn chuyển sang Rust
- Sinh viên CNTT muốn có tài liệu tiếng Việt
